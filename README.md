## 방식

1. 1회차는 코어자바스크립트 기준 키워드를 학습하고 정리한다.
   - 참고 도서: **코어 자바스크립트-모던 자바스크립트 딥다이브-자바스크립트를 말하다**
2. 2회차는 면접형식으로 크루들끼리 서로 질문을 준비하고 대답한다.

- 각자 우테코 닉네임으로 폴더를 만들고, main branch로 PR을 보낸다.
- 파일명 예시 : 01-데이터-타입.md
